# Support

## How to get help

- **Bug reports**: open a GitHub issue using the bug report template.
- **Feature ideas**: open a GitHub issue using the feature request template.
- **Security concerns**: follow the guidance in `SECURITY.md` and avoid filing public issues.

## Before opening an issue

Please include enough detail for maintainers to understand the problem quickly:

- what you are trying to build
- the relevant tree or DSL snippet
- the current selection behavior
- the expected behavior
- any runtime snapshots, screenshots, or failing tests

## Project status

This repository is an early open-source extraction. The API and documentation are still being refined, so maintainers may ask for reproduction cases or suggest narrower problem statements before taking on broader changes.
